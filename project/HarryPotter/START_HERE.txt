✨ 4 ТИПА РЕКОМЕНДАЦИЙ

1. ⭐ ПОПУЛЯРНЫЕ ТОВАРЫ
   - Ранжируются по: (просмотры × 0.3) + (покупки × 0.7)

   - Анализируются заказы пользователей

🐛 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

Проблема: Рекомендации не загружаются
→ Откройте F12 → Console → посмотрите ошибки
→ Проверьте, что подключен скрипт /assets/js/recommendations.js

Проблема: API возвращает 404
→ Убедитесь, что файлы recommendations.js и recommendationsRouter.js в корне
→ Проверьте, что сервер запущен: npm start

Проблема: Стили не применяются
→ Убедитесь, что подключен CSS: /assets/css/recommendations.css
→ Откройте F12 → Network → проверьте статус загрузки

Проблема: Эмодзи не отображаются
→ Добавьте в <head>: <meta charset="UTF-8">

Полная справка: QUICK_START.md → "Типичные проблемы и решения"
